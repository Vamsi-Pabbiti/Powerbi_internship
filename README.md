\# IPL Auction Analysis Dashboard



\## Project Overview

This project analyzes IPL auction data using Power BI.

The dashboard provides insights into team spending, player prices,

and player experience distribution.



\## Tools Used

\- Power BI

\- Excel / CSV Dataset

\- Data Visualization



\## Features

\- Total Auction Spending

\- Average Player Price

\- Top Expensive Players

\- Team Spending Analysis

\- Capped vs Uncapped Player Distribution
